Installation:
To run, extract all files into the same directory and double-click zombie.exe.

If you receive an error saying you are missing msvcxxxx.dll, 
run vcredist_x86.exe and restart zombie.exe.

Controls:
Movement 		- W,A,S,D
Aim			- Mouse
Shoot			- Left mouse button
Grenade			- G or Spacebar
Toggle Fullscreen	- Alt+Enter
